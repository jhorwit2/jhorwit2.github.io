Universal Map

Background

This project was funding by DI2E and run by George Mason's C4I center. The goal of the project was to take the Common Map API specification and develop an application that conforms to the specification.

We implemented a web based solution that gave users the ability to host distributed web applications (widgets) in iFrames as long as the widgets utilized our JavaScript communication library.

My Role

I was the sole developer in-charge of implementing the secure communication library for the widgets. I also wrote the wrapper API to Google V3's library, so that the Google V3 would conform to the CMAPI specification.
